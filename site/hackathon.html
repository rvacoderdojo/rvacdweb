NEVER CODED BEFORE (new people only!)

SC1-SC10

Scratch Challenge Cards

Never ever coded before? That's ok. Just do some of our Scratch challenge cards.

SCRATCH

S1

Hello World! (Scratch)

Create a program in Scratch that asks the user for their name, and tells that user "Hello"

Hint Look under "Sensing" for the "Ask" command and the "answer" variable.

 S2

Animation is Cool!

Draw your own Sprite in Scratch with several costumes (at least 3).

Write a program that uses the sprite and each of it's costumes (this could be a simple animation).

Hint Use "Next Costume" under "Looks"

 S3

I'm So Funny!

Create a program with a character who tells a joke.

Hint Use "Say" under "Looks"

 S4

I Mad Libs

Create a program to do a Mad lib with at least 4 different prompts.

Hint Look under "Sensing" for the "Ask" command and the "answer" variable. You will need a variable for each answer

 S5

Let's Have a Dance Party!

Create a program with characters that dance to music

Hint Look under "Sounds" for commands to play the music, and "Looks" for ways to create dances.

 S6

Bouncy Things

Create a program with an object that bounces off of the edges of the screen.

Hint Look under "Sensing" for the "touching?" logic block and "Motion" for the "x position" and "y position" variables

 S7

My Name Looks Cool!

Create a program that shows your name on the screen using the Scratch letter sprites.

Animate each letter in some way (change color, change size, move around, rotate).

Hint Look under "Looks" for lots of ways to change the appearance of sprites

 S8

My Pet is Hungry

Create a program with a pet. Have the pet follow your cursor to some food and eat the food.

Hint Look under "Motion" for commands to help your pet follow the cursor.

 S9

Mouse Catcher

The Scratch Cat wants to have a little fun! Create a program with a mouse sprite that moves around randomly.

Add a Scratch Cat sprite that you control with the arrow keys chase the mouse around and try to catch it.

Hint Look under "Sensing" for ways to see what key is pressed.

 S10

Puppy Love

Create a program with a puppy. Add a ball to the screen that follows your cursor. Make the puppy chase the ball.

Hint Look under "Motion" for the "go to" command to help the puppy move towards the ball.

 S11

You're a Wizard!

Create a program with a magic wand that follows your cursor around

Make the magic wand add cool random surprises to the screen where ever you click.

Hint Look under "Operators" to generate Random numbers, and use "Events" to send messages to other sprites.

 S12

Story Teller

Tell a quick story using characters and at least 2 different backgrounds.

Hint Look under "Looks" for the "Say" command

WEB PAGES

W1

It's All About ME!

Create a simple web page about yourself.

Use at least 2 different colors (besides black and white) in the creation of the page.

W2

That is My Favorite!

Create a web page about your favorite food or hobby including pictures

Hint Use Google Image search to help you find pictures.

 W3

Summertime is Fun Time

Create a web page about something (or somethings) fun you did this summer.

Add at least 2 pictures from the web (they do not have to be your pictures).

Hint Use Google Image search to help you find pictures.

MULTIPLE LANGUAGES

M1

Hello World!

Create a program in any language other than Scratch (Java, Javascript, Python)

Your program should ask the user for their name, and tell that user "Hello"

Hint In Java use System.out.println() to write to the screen, and the java.util.Scanner to read input from the user

 M2

Guessing Game

Create a program in any language (Java, Javascript, Scratch, Python)

Your program should pick a random number between 0-100.

Keep asking the user to guess the number until they get it right.

If the user guesses wrong tell them if their guess is too high or too low.

Hint In Java use System.out.println() to write to the screen, and the java.util.Scanner to read input from the user

 M3

Simple Math

Create a program in any language (Scratch, Java, Javascript, Python, whatever)

Your program will take 2 numbers as input, and show the result of adding, subtracting and multiplying those 2 numbers.

Hint In Java you can use the commandline to read the 2 numbers (the "args" array variable in main)

 M4

I Know My Multiplication Tables!

Write a program in any language (Scratch, Java, Javascript, Python, whatever)

Your program will take a number as input, and generate the multiplication table for that number up to the number times itself.

For example. If given 5, it will generate the table for 5x1, 5x2, 5x3, 5x4, 5x5

Hint In Java you can use the commandline to read the number (the "args" array variable in main)

 M5

Balloons

Create a Scratch or Greenfoot (Java) program that will add a balloon to the screen where ever the user clicks.

Have the balloon float up to the top and pop when it hits the top.

 M6

Animal Chatter

Create a Java program that demonstrates inheritance by creating an Animal class with a speak() method.

Create 2 or more animal child classes that have different "speak()" actions. Construct each type of animal and make it speak.

Hint Use System.out.println() to write the animal's speech to the screen
