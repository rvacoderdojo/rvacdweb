About RVA Coder Dojo

RVA Coder Dojo is a free, volunteer led, open source resource to help kids, ages 5-17, understand how to become digital producers.

Most children are outstanding digital consumers. Very few understand how the technology they consume is created. We seek to introduce and inspire kids to learn and explore programming, digital and technology design.

Computer science develops studentsâ€™ computational and critical thinking skills and shows them how to create, not simply use, new technologies. This fundamental knowledge is needed to prepare students for the 21st century, regardless of their ultimate field of study or occupation.

We embrace the philosophy of "failing forward". We believe that through both success and failure, we create the foundations for knowledge.

Making Music (and Dancing) with Scratch

Experience Level: Beginner

Recommended Ages: 7-11

Scratch is a very kid friendly programming language for children ages 7-11 that is visually oriented with very little typing required. Scratch allows children to learn the basics of programming in a simple, graphical environment.

During this session the children will be introduced to how to use the Scratch programming environment, and programming concepts such as looping, logic, and variables. At the end of the session each child will have used their own creativity to create a unique musical application.

Computer Science Concepts using NodeJS

Experience Level: Beginner

Recommended Ages: 11-17

Using NodeJS, a Javascript based programming environment, students will be introduced to common computer science concepts such as variables, logic, looping, events, and code libraries which are applicable in many other programming languages. These concepts show up in most programming situations, and a solid understanding of them make learning programming much easier.

During the session we will apply these concepts in the creation of a simple text based game, and if time allows, development of a very basic web server.

Please note: Javascript is a text based language, and requires some ability to type.
