About RVA Coder Dojo

RVA Coder Dojo is a free, open source resource to help kids, ages 5-17, understand how to become digital producers.

Most children are outstanding digital consumers. Very few understand how the technology they consume is created. We seek to introduce and inspire kids to learn and explore programming, digital and technology design.

Scratch

This is a great introductory language for young programmers ages 5 - 11. Scratch introduces basic programming and game making concepts through a simple drag and drop programming paradigm.

Java with Greenfoot

Greenfoot is recommended for older children (ages 10 - 17) who want to work with a more powerful language. Greenfoot helps to introduce the Java langauage which is the starting point for learning how to create Mods in Minecraft.
