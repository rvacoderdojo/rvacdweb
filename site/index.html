About RVA Coder Dojo

RVA Coder Dojo is a free, volunteer led, open source resource to help kids, ages 5-17, understand how to become digital producers.

Most children are outstanding digital consumers. Very few understand how the technology they consume is created. We seek to introduce and inspire kids to learn and explore programming, digital and technology design.

Computer science develops studentsâ€™ computational and critical thinking skills and shows them how to create, not simply use, new technologies. This fundamental knowledge is needed to prepare students for the 21st century, regardless of their ultimate field of study or occupation.

We embrace the philosophy of "failing forward". We believe that through both success and failure, we create the foundations for knowledge.

Create a Simple Game Using Scratch

Experience Level: Beginner

Recommended Ages: 5-11

In this session we will use the Scratch programming language to create a simple video game. Scratch is a very kid friendly programming language for children ages 5-11 that is visually oriented with very little typing required. The children will be introduced to how to use the Scratch programming environment, and programming concepts such as looping, logic, and variables. At the end of the session each child will have a simple working video game that they created.

Introduction to Web Programming

Experience Level: Beginner

Recommended Ages: 5-17

In this session we will learn to apply the same technologies that the web pages you see on Google, Twitter, Instagram, Facebook and YouTube use.

Starting with a basic web page, you will learn to look at it's code, understand what is going on, and add your own changes and enhancements to make it uniquely yours. At the end of this session you will leave with the web page you created and the toolkit we use to continue to explore and learn at home.

Introduction to Java Programming with Greenfoot

Experience Level: Intermediate

Recommended Ages: 10-17

Greenfoot provides a fun and easy to use environment to help children learn the basics of Java programming and Object Oriented design and development. In this session we will start with a very basic game, explore it's logic and code, and add our own enhancements to the game.

While no prior experience with the Java programming language is required, students attending this session should have had some exposure to programming in other languages, and be comfortable with using a text editor (typing, cutting, pasting, and deleting).
